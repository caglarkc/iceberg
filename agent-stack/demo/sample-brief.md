# Mission: Third-Party API Integration Core Service

## Problem
Iceberg X needs a repeatable POC skeleton for third-party SaaS API integrations with OAuth,
webhooks, SDK signature stubs, mock mode, and a small frontend demo page.

## Technical Hints
- Backend: Node.js + Express + TypeScript
- Frontend: React + Vite
- Auth: OAuth 2.0 client credentials
- Endpoints: health, create-resource stub, webhook receiver, SDK signature stub
- Storage: in-memory for POC

## Deliverables
- Runnable repo skeleton
- .env.example with secret names only
- README with setup steps
- TEST_PLAN.md
- HANDOVER.md with known issues and next steps

## Constraints
- Production-ready is out of scope
- Mock mode default
- AI-generated output requires human review
