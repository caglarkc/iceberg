# Demo Screenshots

These text artefacts are captured terminal outputs for the recording/live-demo backup:

- `parse-output.txt`: structured JSON from the demo brief.
- `dry-run-output.txt`: scaffold tree preview with no disk writes.
- `approve-output.txt`: approved scaffold write result.
- `handover-output.txt`: handover package write result.

They are kept as reproducible screenshot substitutes for environments where terminal screenshots are not committed.
