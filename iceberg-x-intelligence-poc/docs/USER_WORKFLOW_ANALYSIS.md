# User Workflow Analysis

## Admin

Needs fast mission creation, consistent fields and visibility into readiness. The POC supports this with AI draft generation, mission save/edit paths and dashboard summaries.

## Mentor

Needs clear review context and a way to publish feedback safely. The POC provides evidence, submission detail, AI review drafts and a human approval gate.

## Intern

Needs explicit deliverables and a simple submission path. The POC provides mission detail, evidence entry and checklist-based submission.

## Leadership

Needs a concise picture of demo readiness and operational risk. The POC provides summary metrics, blocked counts and readiness scoring.
