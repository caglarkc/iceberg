# UX Flow — Lifesycle Zoom Meeting (M3)

## Primary: Valuation Call

1. **Contacts** → select Sarah Mitchell
2. **Contact detail** → property card 14 Oak Lane
3. **Schedule Zoom Meeting** → pre-filled valuation time/topic
4. **Meeting card** appears with status badge
5. **Timeline** → `Zoom meeting scheduled — Valuation call — 14 Oak Lane`
6. **Join in Lifesycle** → slide-over simulated SDK
7. **Leave** → optional webhook → `Meeting ended — 47 min`

## Fallback

Embed unavailable → **Join in Zoom** opens `join_url` in new tab; timeline unchanged.

## Instant Meeting

Contact detail → **Start Instant Meeting** → status `started`, timeline event immediately.
